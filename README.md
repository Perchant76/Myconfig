# PentestOS-config
## This is what it will install:
- OHMYZSH
- OHMYZSH pentest plugin
- OHMYZSH nmap plugin
- TPLINK 755 drivers
## How to install 
`git clone https://github.com/Perchant76/PentestOS-config`

`cd PentestOS-confg`

`chmod +x script.sh`

`./script.sh`

**!!!After install it will actomaticaly reboot so dont be scared about the random reboot ;)**
